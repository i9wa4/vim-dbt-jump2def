# vim-dbt-jump2def

This is a Vim/Neovim plugin jumping to definition in a dbt project.

Read [help](doc/dbt-jump2def.txt) for details.

## 1. Requirements

- <https://deno.com/>
- <https://github.com/vim-denops/denops.vim>

## 2. Installation

Install as a general Vim/Neovim plugin.

## 3. Usage

Execute the command.

```vim
:DbtJump2ModelDef
```
