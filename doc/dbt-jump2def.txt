*dbt-jump2def.txt*	Jumping to definition in a dbt project.

This is a Vim/Neovim plugin jumping to definition in a dbt project.

==============================================================================
COMMAND                                                 *dbt-jump2def-command*

DbtJump2ModelDef
	:DbtJump2ModelDef

	Jump to the definition of the model under the cursor.
